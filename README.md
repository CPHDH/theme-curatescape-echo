# Curatescape Echo
Recommended theme for new Curatescape projects, featuring a clean, modern, typography-centered presentation, with hero images that echo in the background of each item header.

## Companion Plugins
Use of [Curatescape Stories & Tours](https://github.com/CPHDH/Curatescape) and [Curatescape SEO](https://github.com/CPHDH/CuratescapeSeo) companion plugins is strongly recommended. 

## Additional Information
For more information about Curatescape, see [curatescape.org](https://curatescape.org).
